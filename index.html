Quickstart (Python) nexora.init() 
import nexora

nexora.init(
  api_key="YOUR_KEY",
  device_id="ABC123",
  firmware_version="1.2.3"
)

nexora.capture_error(
  "I2C_TIMEOUT",
  message="Sensor read failed"
)
