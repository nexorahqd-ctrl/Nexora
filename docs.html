Quickstart

Nexora SDK is designed to be lightweight and offline-friendly.

Python Install + init 
pip install nexora

import nexora

nexora.init(
  api_key="YOUR_KEY",
  device_id="ABC123",
  firmware_version="1.2.3"
)
Send an error capture_error() 
nexora.capture_error(
  "I2C_TIMEOUT",
  message="Sensor read failed"
)

Event model

Nexora supports 3 core event types in V1:

  • crash
  • error
  • heartbeat
